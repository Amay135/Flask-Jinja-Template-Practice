# Flask Jinja Template Practice

A simple project for learning and practicing Jinja templating in Flask.

## Features

- Flask application setup
- Jinja template rendering
- Passing variables from Flask to templates
- Conditional statements in templates
- Looping with Jinja
- Basic HTML and CSS integration

## Project Structure

```bash
jinja/
│
├── app.py
├── templates/
│   ├── index.html
│   └── ...
├── static/
│   ├── css/
│   └── ...
└── README.md
```

## Installation

1. Clone this repository

```bash
git clone https://github.com/Amay135/jinja.git
cd jinja
```

2. Create a virtual environment (optional)

```bash
python -m venv venv
```

3. Activate the virtual environment

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

4. Install Flask

```bash
pip install flask
```

## Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
